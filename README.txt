****************************************************************

						SoundBoard

****************************************************************
================================================================
****************************************************************

This program is designed to create a customized soundboard.
The user will needs to provide the library of sounds of their
choosing to map onto the soundboard. The soundboard is 
consists of numeric and letter keys. The numeric keys allow
the user to set a sound on repeat. The letter keys play only 
once and will play simultaneously with the repeating numeric
keys. 

Populating the soundboard is the first order of business. Select 
edit mode will allow the user to map a sound from their folder
onto the soundboard. This soundboard only support .wav files.
